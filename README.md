# Pygame Pong

## Rodar o projeto no Linux

### Criar ambiente virtual

```bash
python3 -m venv .venv
```

### Ativar ambiente virtual

```bash
source .venv/bin/activate
```

### Instalar pacotes

```bash
pip install -r requirements.txt
```

### Rodar o projeto

```bash
python3 main.py 
```
